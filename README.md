# calendar-remainder

## Reminder System (Python Project)

## Overview
This project is a simple command-line Reminder System implemented in Python. It allows users to add reminders with a specific date, view all reminders, and search for events by date. All reminders are stored in a text file to ensure data persistence between program executions.

## Features
Add reminders with date
View all reminders
Search reminders by date
Data stored using file handling

## Technologies Used

Python
File Handling
Datetime module

## How to Run

install python (VS Code)
Run the program: reminder system.py

## How It Works

User enters a date in the format YYYY-MM-DD
The program validates the date using datetime module
Reminders are stored in a text file (reminder.txt) in the format:
date,event

When the program starts, users can add, view, or search reminders
All data is read from and written to the file
